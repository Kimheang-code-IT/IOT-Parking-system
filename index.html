Project Idea

  • Smart parking with IoT entry & exit gates
  • Webcam OCR reads license plate at entry
  • Print ticket barcode (plate + invoice + hash)
  • Exit scans barcode → ABA payment → gate opens
  • Live dashboard for staff (parkings, invoices, payment)

Entry Flow


  • Green button → PC webcam OCR plate
  • POST /api/gate/entry/trigger
  • Session PK-* + invoice IN-*
  • Receipt barcode printed / saved
  • Entry servo opens (auto-close 5s)

Payment

  • Exit scan sets fee from parking duration
  • Payment page shows active session (live sync)
  • ABA PayWay KHQR on dashboard
  • Customer pays in ABA app / sandbox confirm
  • Webhook → invoice marked Paid

Exit Flow

  • Scan ticket barcode (plate|invoice|hash)
  • Backend verifies ticket + payment status
  • EXIT_APPROVED → ESP32 exit servo
  • Gate opens · car passes · auto-close
  • Unpaid / invalid → EXIT_DENIED (red LED)

How to Use

Summary
